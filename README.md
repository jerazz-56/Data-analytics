# Sales & Revenue Analysis Dashboard

## Project Overview

The **Sales & Revenue Analysis Dashboard** is a web-based dashboard developed using **HTML, CSS, and JavaScript (Single File Implementation)** to analyze business sales performance and generate meaningful insights.

This dashboard allows users to import sales datasets and visualize important business metrics through interactive charts and KPI cards.

The project demonstrates **data visualization, KPI tracking, and business insight generation** using a responsive dashboard interface.

---

## Objectives

* Import and analyze sales data
* Track business KPIs
* Visualize revenue trends
* Identify top-performing products
* Generate business insights
* Provide interactive filtering and dashboard analysis

---

## Features

### 1. Data Import

Users can upload:

* CSV files (.csv)
* Excel files (.xlsx)

The dashboard automatically reads and processes uploaded datasets.

---

### 2. KPI Tracking

The dashboard calculates:

* Total Sales
* Total Revenue
* Total Orders
* Average Revenue
* Profit Margin %

Formula:

Profit Margin (%) = (Revenue ÷ Total Sales) × 100

---

### 3. Interactive Visualizations

Dashboard includes:

* Revenue Trend (Line Chart)
* Top Performing Products (Bar Chart)
* Revenue by Category (Pie Chart)
* Sales by Region (Bar Chart)
* Profit Margin by Category (Bar Chart)

---

### 4. Filters and Slicers

Users can interactively filter dashboard results using:

* Date Filter
* Product Filter
* Category Filter
* Region Filter

Charts and KPIs update automatically.

---

### 5. Business Insight Generation

The dashboard automatically generates insights such as:

* Best-selling product
* Highest revenue category
* Top sales region
* Profit margin analysis
